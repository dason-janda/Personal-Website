# Personal Portfolio Website

This repository contains the source code for my professional personal website

[**View My Website Here**](https://d00476471.github.io/Personal-Website/) 

## Project Overview
The goal of this project was to design and develop a responsive, accessible, and modern personal portfolio from scratch. It serves as a way to showcase some of my technical skills and have somthing I can send future employers

### Key Features:
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing using CSS Media Queries.
- **Modern Layouts:** Utilizes CSS Flexbox and Grid for precise alignment and structure.
- **Navigation:** A functional, intuitive navigation system across all sub-pages.
